# 🚀 Guia de Início Rápido

## Como Começar Imediatamente

### 1. Abrir o Site

**Opção 1 - Localmente:**
```bash
1. Abra a pasta do projeto
2. Dê duplo clique em index.html
3. O site abrirá no seu navegador padrão
```

**Opção 2 - Via Terminal:**
```bash
# Windows
start index.html

# Mac
open index.html

# Linux
xdg-open index.html
```

**Opção 3 - Com Servidor Local (recomendado):**
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000

# Depois acesse: http://localhost:8000
```

### 2. Explorar o Site

**Fluxo Recomendado:**

1. 🏠 **Início** → Entenda a missão
2. 📚 **Informações** → Saiba mais sobre o tema
3. 💬 **Recursos** → Conheça onde obter ajuda
4. 🔔 **Denúncia** → Quando estiver pronto

### 3. Fazer uma Denúncia

**Passo a Passo:**

1. Vá para "Fazer Denúncia"
2. Selecione o tipo(s) de abuso
3. Preencha as datas
4. Descreva a situação (detalhado mas sem identificação)
5. Verifique as confirmações
6. Clique "Enviar Denúncia Anonimamente"
7. **Salve o ID gerado!**

**Dicas Importantes:**
```
❌ NÃO mencione:
   - Nomes de pessoas
   - Nome da empresa
   - Cidade específica
   - Datas exatas
   - Seu cargo específico
   - Seu nome ou contato

✅ MENCIONE:
   - Tipo de abuso
   - Como se comportava
   - Consequências para você
   - Quando começou (mês/ano)
   - Se havia testemunhas
```

## Troubleshooting

### Problema: Página não carrega

**Solução:**
1. Atualize a página (F5 ou Ctrl+R)
2. Limpe cache (Ctrl+Shift+Delete)
3. Tente outro navegador
4. Verifique conexão de internet

### Problema: Formulário não funciona

**Solução:**
1. Verifique se preencheu todos campos
2. Abra Console (F12) para ver erros
3. Tente desabilitar extensões de navegador
4. Reinicie o navegador

### Problema: Denúncia não foi salva

**Solução:**
1. Verifique se localStorage está ativado
2. Veja se há espaço disponível
3. Abra Console e procure por erros
4. Tente em navegador privado

### Problema: Perdi meu ID de denúncia

**Dica:**
- Se salvou em localStorage, está seguro
- Abra Console (F12) e procure
- Se não encontrar, pode fazer nova denúncia

## Configuração do Navegador

### Recomendações

**Para Máxima Privacidade:**

1. **Chrome/Chromium:**
   - Usar modo Incógnito
   - Limpar dados ao sair
   - Desativar cookies

2. **Firefox:**
   - Usar modo Privativo
   - Políticas de rastreamento: Restritivo
   - Limpeza automática ao sair

3. **Safari:**
   - Usar navegação privada
   - Bloqueio de rastreadores: Ativo
   - Limpar dados ao sair

4. **Edge:**
   - Usar InPrivate
   - Proteção contra rastreamento: Estrita

## Perguntas Frequentes

**P: É realmente anônimo?**
R: Sim! Você não precisa se identificar em nenhum momento.

**P: Meus dados vão para o governo?**
R: Não (na versão atual). Ficam apenas no seu navegador.

**P: Posso fazer múltiplas denúncias?**
R: Sim, quantas precisar. Cada uma terá seu ID.

**P: Alguém pode rastrear minha denúncia?**
R: Não. Sem informações de IP ou localização.

**P: Posso acessar minhas denúncias depois?**
R: Sim, no mesmo navegador, mesma máquina.

**P: O que fazer se minha empresa descobrir?**
R: Denúncia sem identificação = impossível achar culpado. Mas procure ajuda jurídica.
