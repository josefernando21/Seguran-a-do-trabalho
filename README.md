# 🛡️ Segurança do Trabalho - Canal de Denúncias Anônimas

Um site dedicado a fornecer um espaço seguro e confidencial para trabalhadores denunciarem abusos psicológicos no ambiente corporativo de forma anônima.

## 📋 Visão Geral

Este projeto é uma plataforma web completa que visa:

- **Conscientização**: Informar sobre psicologia no trabalho e tipos de abuso
- **Denúncia Segura**: Permitir que trabalhadores façam denúncias sem se identificar
- **Apoio**: Fornecer recursos e orientações para vítimas de abuso psicológico
- **Confidencialidade**: Garantir total sigilo e proteção de dados

## 🎯 Objetivos

1. Criar um canal anônimo para denúncias de abusos psicológicos
2. Educar sobre saúde mental no trabalho
3. Fornecer informações sobre direitos trabalhistas
4. Conectar vítimas com recursos de apoio
5. Documentar e visibilizar o problema

## 🔒 Segurança e Confidencialidade

### Garantias Implementadas:

1. **Anonimato Total**
   - Nenhuma informação pessoal é obrigatória
   - Nenhuma rastreabilidade de IP
   - Sem necessidade de login

2. **Armazenamento Seguro**
   - Dados armazenados localmente (localStorage)
   - Sem transmissão para servidores externos (versão estática)
   - Possibilidade de encriptação future

3. **Políticas de Privacidade**
   - Dados nunca são compartilhados
   - Sem cookies de rastreamento
   - Sem analytics invasivas

## 🚀 Como Usar

### Abrindo o Site

1. Faça download de todos os arquivos
2. Abra `index.html` em seu navegador
3. Navegue pelas páginas

### Fazendo uma Denúncia

1. Acesse a página "Fazer Denúncia"
2. Preencha o formulário com as informações
3. Evite colocar dados que possam identificar você
4. Clique em "Enviar Denúncia Anonimamente"
5. Guarde o ID gerado para referência futura

### Consultando Informações

- Visite a página "Informações" para aprender sobre psicologia no trabalho
- Acesse "Recursos" para encontrar ajuda profissional
